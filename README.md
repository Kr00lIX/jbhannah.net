# jbhannah.net

Homepage and weblog of Jesse B. Hannah.

## Deployment

The site is [compiled][] by [Jekyll][] when pushed to [Heroku][]. The
site on Heroku runs on a single dyno with no addons and the
[`user-env-compile`][] Labs feature.
